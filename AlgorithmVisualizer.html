Data Structures and Algorithms Visualizer Website

UX CASE STUDY

College project for subject User Experience Design (UI/UX): Designed and Developed a website which enables users to visualize and understand Data structures and algorithms.

Project Type: Team (Shruti Adhav, Vaishnavi Gudulkar, Anjani Kadam)

Contribution: UX research, Ideate, Wireframes, Low-fidelity Prototype, Mockups, High Fidelity Prototype

Duration: 6 months

Tools used: React, CSS, Figma

Objective

The objective of Data Structures and Algorithms Visualizer website is to provide an interactive and engaging platform for all users, regardless of their background i.e. Computer Science or non-Computer Science background. The website aims to simplify learning by providing visualization -- step by step animation of the concepts. Users can also input their own data, enhancing engagement. The Data Structures and Algorithms Visualizer website seeks to help users comprehend and visualize concepts, making learning fun and easy.

Problem Statement

Multiple students and professionals often struggle to understand the complex Data structures and algorithms because of no real-time visual representation and abstract concepts. The challenge is to create a website that would allow users to easily visualize complex data structures and algorithms, making learning more accessible and intuitive. Hence through the website complex data structures and algorithms can be easily visualized and understood as well as make learning engaging and enjoyable.

Project Goals

  • Help users visualize step by step execution of data structures and algorithms as well as interact with the data structures and algorithms to facilitate a better and clear understanding of the complex data structures and algorithms.
  • Enable the user to interact with the data structures and algorithms by tweaking the inputs and see how the input changes.
  • Provide users with enjoyable and captivating learning experience by using animations, color coding and interactive elements which will hold the user’s attention.
  • Allow users to compare different algorithms easily.

Research

In this project we started off with both qualitative and quantitative research by creating a questionnaire, which we distributed through google forms. This allowed us to reach a wider audience easily. The data provided a clear understanding of what the user’s pain points and needs were. Based on the findings then we created user personas to represent the different user types. This ensured that the website would really help to create a more user-centered and impactful design for the users.

Questionnaire

Questions for all users
  • On a scale of 1 to 4, rate your proficiency in complex data structures and algorithms?
  • Which data structures and algorithms do you find difficult to visualize? (Due to time constraints we provided users with a selection of complex algorithms from which they could choose)
  • Do you currently use any platforms for visualizing data structures and algorithms?
For users using a data structures and algorithms visualizer or who have used in the past
  • Which platforms do you currently use to visualize data structures and algorithms?
  • What challenges have you faced with existing visualizers?
  • What additional features would you like in a data structure and algorithm visualizer?
For users not using a visualizer
  • What factors have deterred you from using visualization tools?
  • Would you be interested in using a visualizer for understanding data structures and algorithms?
  • What features do you expect from a data structures and algorithm visualizer?

Complex Data structure and Algorithms Insight

Based on the data collected we recognized that users found it difficult to visualize the following data structures and algorithms: Linked lists, Searching algorithm, Sorting algorithms Queue and Stack. Hence we decided to focus on these five data structures and algorithms for our project.

Personas

Ankita Patil is 19 year old Computer Science student from Pune, India. She has intermediate proficiency in data structures and algorithms. She had previously used a visualization website before however she faced some challenges. The platform did not allow her to input her own data and it lacked interactive and color coding elements, making everything monotonous. Additionally, website did not provide an easy way to compare different data structures and algorithms. Ankita desires a more enjoyable learning experience and seeks visualization website that offers clear, interactive tools to help better understand the complex data structures and algorithms.

John Patel is a 28 year old Mechanical Engineer from Mumbai and spent the last 3 years working in a large corporation. Recently he has become interested in the role of software and in automating engineering tasks. His desire is to transition from mechanical engineering to software engineering within his company. Being a beginner John faces multiple hurdles along his journey. He finds it difficult in writing efficient code and understanding the complex data structures and algorithms concepts. John is in need of a visualization tool which will allow him to easily interact and understand the execution of complex data structure and algorithms. He should be able to put in his own values. John’s goal is to build a strong foundation in software development and apply his knowledge to real time projects.

Heuristic Evaluation

A heuristic evaluation was carried out in class in which other group members provided valuable feedback on Data Structures and Algorithms website we created. One of the members suggested to include a language selection option, which help website to reach a broader audience. Additionally, they proposed to make the website fully responsive as for now the website is only applicable for desktops. By making the website responsive it will enhance accessibility and usability across different devices.

Iterations based on Heuristic Evaluation

Based on the feedback from our classmates, we executed a language selection option in the high-fidelity prototype of Data Structures and Algorithms Visualizer. Incorporating this feature into the website along with making the website fully responsive across all devices is still in progress.

Takeaway

The website Data Structures and Algorithms allows a visual learning experience. It helps to simplify complex topics like Linked Lists, Queues, Stacks and Sorting as well as Searching algorithms. We followed a user-centric approach, focusing on the need of the users. Designing and developing this website was a wonderful experience, especially knowing that it would benefit a broad range of users. The insights and feedback from classmates were invaluable it helped us to create a website to be more user-focused and accessible. Improvements are still in progress however I’m personally really eager to continue and keep refining and iterating on designs to ensure that website remains intuitive and accessible to user needs.
